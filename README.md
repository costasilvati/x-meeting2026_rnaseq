# x-meeting2026_rnaseq
Repositório inclui o material apresentado no X-meeting 2026, realizado em Garibaldi - Rio Grande do Sul, de 9 a 12 de junho de 2026. O material compõe o conteúdo do curso "Expressão Diferencial de Genes com dados de RNA-Seq utilizando a linguagem R", elaborado e ministrado por Juliana Costa Silva e Fabrício Martins Lopes.
